Dados

Os dados são obtidos através de duas rotas disponibilizadas pela Agrotools através de outra API REST. São elas:

  • Analysis
  • List Territories


Analysis

A primeira parte dos dados são análises dos territórios expondo critérios de legalidade.

Cada relatório é identificado pelo seu código único.


List Territories

A segunda parte contém características do território como nomes dos proprietários, município, estado, nível, etc.

Também obtidos através do código único de cada território.


Banco de dados

O banco de dados contruído contém ambas as partes separadas e unidas, totalizando 6 tabelas:

  • analysis raw
  • list territories raw
  • full data raw
  • full data stage

Nota-se a presença separação dos dados brutos (raw) e refinados (stage).
